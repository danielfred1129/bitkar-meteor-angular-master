# meteor-angular
Starter in Meteor &amp; Angular


## Install

 1. Install Meteor, https://www.meteor.com/install

 2. Copy project
    
    $ git clone https://github.com/danielfred1129/meteor-angular.git
    
    $ cd meteor-angular

 3. Update meteor & npm

    $ meteor update
    
    $ meteor npm install


## Run

   $ meteor run
