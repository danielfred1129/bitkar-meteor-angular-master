# bitkar-meteor-angular
Bitkar in Meteor &amp; Angular


## Install

 1. Install Meteor, https://www.meteor.com/install

 2. Copy project
    
    $ git clone https://github.com/arangde/bitkar-meteor-angular.git
    
    $ cd bitkar-meteor-angular

 3. Update meteor & npm

    $ meteor update
    
    $ meteor npm install


## Run

   $ meteor run
